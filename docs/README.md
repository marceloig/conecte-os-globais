## Stack Tecnológica

```
Frontend:  React 19 · TypeScript · Vite · React Flow · Radix UI · Tailwind CSS
Backend:   FastAPI · Python 3.11+ · Pydantic · HTTPX
Database:  Neo4j (Banco de Grafos)
Scraping:  Scrapy
Imagens:   TMDB API
Deploy:    Docker · Docker Compose · AWS (SSM Port Forwarding)
```
